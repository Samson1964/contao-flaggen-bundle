# Flaggen für Contao

Die Erweiterung stellt ein Inserttag flagge zur Verfügung, welches Flaggensymbole im Format 4:3 anzeigt. Die anzuzeigende Flagge muß mit dem dreistelligen ISO-Code des Landes bestimmt werden.

## Kurzanleitung

Beispiel für die Flagge von Deutschland mit einer Breite von 50 Pixel:

````
{{flagge::GER,50}}
````

Wenn die Breitenangabe weggelassen wird, erscheint die Flage mit 20 Pixel Breite. Beispieleinbindung für Frankreich:

````
{{flagge::FRA}}
````
